# fmm_sofr
functional mixed model in scalar on functional regression with repeated outcomes

This Rmarkdown file provides code and simulated examples for implementing the functional mixed model
in scalar on functional regression with repeated outcomes, proposed in the manuscript titled
"A Functional Mixed Model for Scalar on Function Regression with Application to a Functional MRI Study",
authored by Wanying Ma, Luo Xiao, Bowen Liu from NC State University and Martin Lindquist from Johns Hopkins
University.
